Problem set: gripping inside the window

Six problems on the grip-force window between slip and bruise - computing the force, sizing the actuator to land inside the window, reasoning about compliance, and defending the independent safety relief. Every quantitative answer is a single number you can check. Attempt them before revealing the worked solutions.

Realizes CEC-02, EI-09 Problems 6 Estimated 60-90 min
\[ F = P\,A \qquad \text{and} \qquad F_{\mathrm{slip}} \le F_{\mathrm{grip}} \le F_{\mathrm{bruise}} \]

Grip force is cylinder pressure times piston area. A successful grasp holds the fruit (above the slip bound) without damaging it (below the bruise bound). The design job is to land the operating force inside that window with margin on both sides.

Solution 1 - Compute the grip force
\[ F = P\,A = 450\times10^{3}\ \mathrm{Pa} \times 1130\times10^{-6}\ \mathrm{m^2} = 508\ \mathrm{N} \]

Inside the window: 250 <= 508 <= 800. Margin to slip = 508 - 250 = 258 N; margin to bruise = 800 - 508 = 292 N. A well-centered operating point.

Solution 2 - Map the window to pressure
\[ \begin{aligned} P_{\mathrm{slip}} &= F_{\mathrm{slip}}/A = 250/1130\times10^{-6} = 221\ \mathrm{kPa} \\ P_{\mathrm{bruise}} &= F_{\mathrm{bruise}}/A = 800/1130\times10^{-6} = 708\ \mathrm{kPa} \\ \text{band width} &= 708-221 = 487\ \mathrm{kPa} \end{aligned} \]

These match the ratified P_slip ~221 kPa and P_bruise ~708 kPa - confirming the ~38 mm bore is consistent with the force window. You have a 487 kPa operating band to place the grip inside.

Solution 3 - Size the actuator
\[ \begin{aligned} A &= F/P = 525/500\times10^{3} = 1.05\times10^{-3}\ \mathrm{m^2} = 1050\ \mathrm{mm^2} \\ d &= 2\sqrt{A/\pi} = 2\sqrt{1050/\pi} = 36.6\ \mathrm{mm} \end{aligned} \]

Slightly smaller than the ratified ~38 mm. The direction is right: a higher supply pressure (500 vs the ~465 kPa the 38 mm bore needs for 525 N) lets a smaller piston produce the same force, since \(F = P\,A\) trades pressure against area.

Solution 4 - Compliance and force error
\[ \begin{aligned} \text{(a)}\quad dF = k_{\mathrm{eff}}\,\delta:\ & \text{at } 2\ \mathrm{mm} \to 20\times2 = 40\ \mathrm{N};\ \ \text{at } 5\ \mathrm{mm} \to 20\times5 = 100\ \mathrm{N} \\ \text{(b)}\quad k_{\max} &= dF_{\max}/\delta = 275/5 = 55\ \mathrm{N/mm} \end{aligned} \]

The ratified 20 N/mm is well under the 55 N/mm ceiling - at 5 mm of overshoot it adds only 100 N, comfortably inside the window. (c) A softer gripper turns the same position error into a smaller force error, so more of the position-error range still lands inside the force window - compliance widens the effective tolerance.

Solution 5 - Defend the safety relief (model answer)

(a) The software clip depends on the force sensor, the controller, and the actuator all working; if the sensor drifts, the controller hangs, or a command is corrupted, the clip fails and cylinder pressure can rise to full supply - crushing the fruit. That entire chain is a single failure domain. (b) An independent mechanical relief valve set at the bruise pressure (~708 kPa) caps the cylinder pressure regardless of software state, so grip force cannot exceed ~800 N even with total controller failure. (c) It is set at the bruise bound because the relief's job is to prevent damage - exceeding slip only loses the fruit (a performance miss), while exceeding bruise destroys it (a quality/safety failure). This two-domain independence is the point of EI-09/EI-11: the safe bound must not share a failure path with the thing it protects against.

Solution 6 - A firmer cultivar
\[ \begin{aligned} \text{(a)}\quad & \text{window } [300, 900]\ \mathrm{N} \to \text{width } 600\ \mathrm{N}\ (\text{was } 550) \\ & P_{\mathrm{slip}} = 300/1130\times10^{-6} = 266\ \mathrm{kPa};\ \ P_{\mathrm{bruise}} = 900/1130\times10^{-6} = 797\ \mathrm{kPa} \\ \text{(b)}\quad & \text{op point } 508\ \mathrm{N} \in [300,900]?\ \text{yes};\ \text{slip margin } 508-300 = 208\ \mathrm{N},\ \text{bruise margin } 900-508 = 392\ \mathrm{N} \\ \text{(c)}\quad & \text{re-center at mid} = (300+900)/2 = 600\ \mathrm{N} \to P = 600/1130\times10^{-6} = 531\ \mathrm{kPa} \end{aligned} \]

The window shifted up and widened, so the old operating point still holds but now sits closer to slip (208 N vs 258 N margin). Raising the supply to 531 kPa re-centers the grip at 600 N, restoring symmetric margin. The lesson: a window is defined by the fruit, not the actuator - re-characterize the bounds when the crop changes, then re-center.
